# Risks

**Concurrency risks** — *Low (was High), by design.* The checkpoint's High risk
(optimistic locking over aliased reads) is answered by D1; the undefined-seq risk
by D2 (allocator authority, contiguous 1-based, `seq=0` sentinel, admission
rejection of pre-assigned seq). Residual, accepted and documented: OS-thread
safety is out of scope (single logical writer per `Engagement`, consistent with
M1-Decomposition's exclusions). Watch during implementation: the R/W owner
matrix stays **data-only** (D6) — any temptation to invent roles mid-M1.7 is
architecture creep and must return to the design.

**Persistence risks** — *Low, deferred by design.* Zero IO in the state package
(re-verified). M1.7's contribution is the atomic-commit invariant M1.8 must
preserve; durable crash-ordering is explicitly M1.8 design scope. One guard
already identified for M1.8: never persist the empty-projection placeholder
state (`EngagementId("")`/epoch metadata).

**Replay risks** — *Low.* Replay-as-composition is property-tested since M1.5;
D2's boundary integrity check (`ReplayIntegrityError`) closes the
caller-supplied-order hole; the `projection_version` policy governs the D4 bump
(1→2) with a required CHANGELOG migration note. Duplicate-seq detection exists at
the only boundary where duplicates can enter (external logs).

**Versioning risks** — *Low.* `state_version` becomes log-derived in the fold
(D4), making the invariant triangle (`current_version == state_version ==
len(log)`) testable and replay-safe. The bump is the **first controlled
exercise** of the versioning policy — execute it exactly per
`docs/architecture/projection-versioning.md`, since it sets the precedent.

**Knowledge Layer risks** — *None for M1.7.* The vault is empty, Graphify is
path-scoped away from source, and the state layer has zero knowledge-layer
imports (verified: 0 grep hits). Known M2 items (frontmatter typed edges,
note-relative wikilink resolution, manual reindex trigger, static-graph MCP
restart) are recorded in the Graphify review and have no path into M1.7.
