# Value Chain Digitization

**Domain(s).** [[domains/digital-transformation]]

**Purpose.** Map the value chain and identify where digitization creates cost savings, revenue uplift, or service improvements.

**When to use.** When scoping a digital transformation and prioritizing where to invest across the value chain.

## Diagnostic questions
- Which value chain steps have the highest digital value potential relative to current state?
- Where is process digitization the right answer vs. business model digitization?

## Success metrics
- Digital value as % of total transformation opportunity
- Cycle-time reduction in digitized steps

## Common risks
- Digitizing inefficient processes without fixing the underlying process design first

## Common mistakes
- Focusing on digitization volume without a value case for each initiative
