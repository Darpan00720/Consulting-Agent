# McKinsey 7S Framework

**Domain(s).** [[domains/organizational-design]]

**Purpose.** Assess alignment across seven organizational elements (Strategy, Structure, Systems, Shared Values, Style, Staff, Skills) to diagnose organizational fit.

**When to use.** When diagnosing why an organizational change is not sticking or why performance is misaligned with strategy.

## Diagnostic questions
- Are all seven elements aligned and mutually reinforcing?
- Which elements are most misaligned with the current strategy?

## Success metrics
- 7S alignment assessment score by element
- Number of misaligned elements vs. prior assessment

## Common risks
- Treating 7S as a design framework when it is primarily a diagnostic framework

## Common mistakes
- Addressing structural elements (hard Ss) without addressing cultural elements (soft Ss)
