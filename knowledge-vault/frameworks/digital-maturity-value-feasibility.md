# Digital Maturity & Use-Case Value/Feasibility

**Domain.** [[domains/digital-transformation]]

**Purpose.** Baseline digital maturity, map value pools, and sequence use cases by value × feasibility.

**When to use.** Prioritizing a digital transformation roadmap.
**When not to use.** A point automation problem, not a transformation.

## Logic
Maturity baseline → value pools by value-chain step → use-case prioritization → capability/operating-model gaps.

## Diagnostic questions
- Where do the value pools sit?
- Is the data and capability ready?

## Success metrics
- Digitization ROI
- Cycle-time
- Adoption
