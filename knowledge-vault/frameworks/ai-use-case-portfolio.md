# AI Use-Case Portfolio

**Domain.** [[domains/ai-strategy]]

**Purpose.** Prioritize AI use cases by value × feasibility × data-readiness; value-led, data-feasible, governed.

**When to use.** Prioritizing an AI portfolio and operating model.
**When not to use.** When there is no data foundation or value hypothesis.

## Logic
Value mapping → use-case prioritization (value × feasibility × data-readiness) → build/buy → operating model + governance.

## Diagnostic questions
- Where is the value?
- Is the data ready?
- Build or buy?

## Success metrics
- Use-case ROI
- Time-to-value
- Adoption
